## Qu’est-ce qu’une prop firm de paris sportifs ?

Une "Prop Firm" (pour Proprietary Trading Firm) est une entreprise qui met **son propre capital à la disposition de parieurs sportifs talentueux**.  

Le concept est simple :  
- La société vous finance  
- Vous pariez avec l'argent de l'entreprise  
- En cas de gains, vous partagez les bénéfices (généralement 70% à 90% pour vous)  
- En cas de perte, c'est l'entreprise qui assume le risque  

Pour prouver vos compétences, la Prop Firm demande de passer un **Challenge** ou **Phase d'évaluation**.
